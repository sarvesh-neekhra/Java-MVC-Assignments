# Java MVC Assignments

## 📌 Overview

This repository contains my Java MVC (Model-View-Controller) assignments completed as part of my learning in Advanced Java. The projects demonstrate the implementation of the MVC architecture using Java technologies such as Servlets, JSP, JDBC, and MySQL.

The objective of this repository is to practice building structured, maintainable, and scalable web applications while following the MVC design pattern.

---

## 🚀 Technologies Used

- Java
- Servlet
- JSP
- JDBC
- MySQL
- Apache Tomcat
- Eclipse IDE
- Maven (where applicable)

---

```
ProjectName/
```

## 🎯 Learning Objectives

- Understand the MVC architecture.
- Separate business logic from presentation.
- Connect Java applications with MySQL.
- Perform Create, Read, Update, and Delete (CRUD) operations.
- Build dynamic web applications using Servlets and JSP.

---

## 📖 How to Run

1. Clone the repository.

```bash
git clone https://github.com/sarvesh-neekhra/Java-MVC-Assignments.git
```

2. Import the project into Eclipse.

3. Configure Apache Tomcat.

4. Create the required MySQL database.

5. Update the database configuration if necessary.

6. Run the project on the Tomcat server.
